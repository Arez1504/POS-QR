# 🔍 Phân Tích Kiến Trúc Dự Án QR-CODE POS

---

## 1. Mục Đích Chính Của Dự Án

Đây là một **Hệ thống Bán hàng tại quầy (POS - Point of Sale)** kết hợp công nghệ **quét mã vạch/QR Code**, được thiết kế cho các cửa hàng bán lẻ nhỏ tại Việt Nam.

### Điểm nổi bật:

| Tính năng | Mô tả |
|---|---|
| 🛒 **Bán hàng POS** | Tạo đơn hàng, tính tiền, xuất hóa đơn tại quầy |
| 📱 **Biến Điện thoại → Máy quét** | Dùng camera ĐT quét mã vạch gửi real-time về màn hình POS qua WebSocket |
| 💳 **Thanh toán VietQR** | Tạo mã QR thanh toán chuyển khoản ngân hàng theo chuẩn VietQR |
| 🏆 **Tích điểm thành viên** | Hệ thống loyalty: tích lũy & tiêu điểm khi mua hàng |
| 📊 **Báo cáo & Thống kê** | Dashboard doanh thu, biểu đồ, top sản phẩm, hiệu suất nhân viên |
| 🔄 **Quản lý ca làm việc** | Mở ca/đóng ca, đối soát két tiền mặt |
| 📦 **Quản lý kho** | Nhập hàng, theo dõi tồn kho, lịch sử xuất/nhập |

> Tóm gọn: Đây là một **ứng dụng POS full-stack hoàn chỉnh**, từ quản lý sản phẩm, bán hàng, kho hàng, nhân sự cho đến báo cáo tài chính — tất cả tích hợp quét mã vạch/QR và thanh toán không tiền mặt.

## 2. Các Công Nghệ/Thư Viện Chính

### 🏗️ Kiến trúc tổng thể: **3-Tier Architecture**

    [Frontend - Browser/Mobile]
        React 19 + TypeScript
        Vite (Build Tool)
        TailwindCSS (Styling)
            |
            |--- HTTP REST API (Axios) --->  [Backend - API Server]
            |--- WebSocket (real-time) --->      Node.js + Express 5
                                                 Socket.IO Server
                                                     |
                                                     |--- mysql2/promise ---> [Database]
                                                                               MySQL / MariaDB

### 🎨 Frontend (Giao diện)

| Thư viện | Phiên bản | Vai trò |
|---|---|---|
| **React** | 19.x | Framework UI chính, xây dựng giao diện SPA |
| **TypeScript** | 6.x | Kiểm tra kiểu dữ liệu tĩnh, tăng độ tin cậy code |
| **Vite** | 8.x | Build tool siêu nhanh, HMR (Hot Module Reload) |
| **TailwindCSS** | 3.x | Framework CSS utility-first, styling nhanh |
| **React Router DOM** | 7.x | Điều hướng giữa các trang (SPA routing) |
| **Axios** | 1.x | HTTP client gọi API backend |
| **Socket.IO Client** | 4.x | Kết nối WebSocket real-time (nhận mã vạch từ ĐT) |
| **html5-qrcode** | 2.x | Quét mã vạch/QR bằng camera thiết bị |
| **JsBarcode** | 3.x | Tạo hình ảnh mã vạch EAN/UPC để in nhãn |
| **qrcode** | 1.x | Tạo hình ảnh mã QR (VietQR thanh toán) |

| Công nghệ | Vai trò |

| **MySQL / MariaDB** | Lưu trữ dữ liệu: sản phẩm, đơn hàng, users, ca làm việc, khách hàng... |

### 📂 `database/` — Cơ sở dữ liệu

| File | Ý nghĩa |
|---|---|
| schema.sql | **Schema gốc** — tạo DB `pos_qr`, bảng `users`, `products`, `orders`, `order_items`, `shifts` + tài khoản admin mặc định |
| migration_v2_customers.sql | **Migration v2** — thêm bảng `customers` (thành viên, tích điểm loyalty) |
| migration_v3_activity_logs.sql | **Migration v3** — thêm bảng `activity_logs` (audit log) |
| workbench_queries.sql | Tập hợp câu SQL mẫu để test/debug trên Workbench |

> ⚠️ QUAN TRỌNG: Phải chạy theo thứ tự: schema.sql → migration_v2 → migration_v3 để DB hoạt động đầy đủ.
